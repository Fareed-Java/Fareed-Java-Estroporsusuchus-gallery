# Estroporsusuchus Gallery

A simple photo gallery website inspired by paleoart galleries.

## How to Use

1. Add your photos to the appropriate subfolders in the `images` folder:
   - `images/dinosaurs/` for dinosaur images
   - `images/marine-reptiles/` for marine reptile images
   - `images/pterosaurs/` for pterosaur images
   - `images/kpg/` for other prehistoric images
2. Run `python3 generate_images.py` to update the gallery with new images.
3. Open `index.html` in a web browser to view the gallery.

## Features

- Responsive grid layout
- Lightbox for viewing images
- Load more functionality for large galleries
- Background music player
- Easy to add new images dynamically

## Customization

- Edit `styles.css` to change the appearance.
- Add more sections or categories as needed.
- Modify `generate_images.py` to include additional image formats if needed.
